Crafthead

A radically fast Minecraft avatar service.

Why use Crafthead?

  • Runs on the edge – runs fast everywhere, using Cloudflare's 200+ datacenters worldwide
  • Serverless – not only does it run fast everywhere, it scales with zero intervention
  • Ridiculously fast – depending on if the skin has been cached and the desired avatar size, it can be more than 2x as fast as other services
  • Easy to move to – the URL layout is 100% compatible with Minotar's, so it's a quick and easy switch

Getting started

Crafthead supports lookups by UUID (dashed or not) and usernames and supports rendering avatars or retrieving skins. The URL layout is compatible with Minotar, making migration simple.

Performance tip: Use UUIDs when possible. Using a username requires an extra API call, which may not be cached!

Fetch an avatar

GET https://crafthead.net/avatar/652a2bc4e8cd405db7b698156ee2dc09

Need something bigger?

GET https://crafthead.net/avatar/652a2bc4e8cd405db7b698156ee2dc09/128

Got a dashed UUID? We will remove them for you.

GET https://crafthead.net/avatar/069a79f4-44e9-4726-a5be-fca90e38aaf5

Fetch a player's skin

GET https://crafthead.net/skin/652a2bc4e8cd405db7b698156ee2dc09
